# Netflix-Clone
A front-end web development project that replicates the visual design and layout of the official Netflix homepage. Built entirely with HTML5 and CSS3 to practice semantic layouts.

🎯 Learning Objectives

This project helps in understanding:

HTML page structure

CSS Flexbox and Grid layouts

Responsive web design

CSS positioning and styling

Creating modern landing pages
